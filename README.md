# 🎓 Student Performance Prediction

## 📌 Project Overview
This project analyzes student academic performance using data analysis and machine learning techniques.  
The goal is to understand the factors affecting student scores and build a predictive model for performance evaluation.

The notebook is fully executable and previewable directly on GitHub.

---

## 🧠 Problem Statement
Educational institutions often need early indicators of student performance to provide timely support.  
This project aims to:
- Analyze student data
- Identify key performance factors
- Predict student outcomes using ML models
